# 🌤️ Weather App - Приложение погоды

## Описание

Консольное/десктопное приложение для получения актуальной информации о погоде в любом городе мира.

## Возможности

- 🌍 Получение погоды по названию города
- 🌡️ Температура, влажность, давление
- 💨 Скорость ветра
- 🕐 Время восхода и заката солнца
-  Детальная информация о погодных условиях

## Технологии

- Python 3.10+
- Requests (HTTP-запросы)
- OpenWeatherMap API
- Tkinter (GUI, если используется)
